Projeto Didático: Força Bruta e Mitigação (Hipotético)

Aviso de Segurança (Crucial)

Este projeto é estritamente educacional e defensivo. Todos os testes devem ser realizados em um ambiente de laboratório isolado (VMs) sob seu controle, usando targets intencionalmente vulneráveis (ex.: Metasploitable, DVWA). Jamais execute técnicas de ataque contra sistemas, redes ou ativos de terceiros sem permissão explícita.

Objetivo

Simular ataques de força bruta e password spraying em um ambiente controlado para entender vetores, registrar evidências e, principalmente, documentar medidas de mitigação e defesa.
